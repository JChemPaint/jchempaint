The File menu

New

Applet: The drawing is cleared. Use undo to get your drawing back. Application: A new empty frame is generated.

Open

Opens a structure read from one of the following file formats (the type is determined from the files extension). In the application, a new frame will be opened, in the applet the current structure is replaces (use undo to get the old content back. Possible formats are explained here.

Save

Saves the current structure in the currently edited file. If you are working on a new structure, you will be forwarded to Save As.

Save As...

Saves the current structure in a file you can specify. Possible formats are explained here.

Save as image..

This menu option can be used to generate images of the structure drawn. These formats can be saved: BMP, JPEG, SVG and PNG.

Print

This prints the current drawing.

Close (not for Applet)

This quits the JChemPaint frame you are working in. You will be warned if there are unsave data.

Exit (not for Applet)

This quits all JChemPaint frames. You will be warned if there are unsave data.
