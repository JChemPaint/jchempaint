How to start JChemPaint

If you haven't started JChemPaint yet, you can do this using a Java virtual machine, by typing 

java -jar jchempaint.jar
in the directory where the JCPCDK_Home/jchempaint.jar is located, or, on Windows, by double clicking on the jchempaint.jar archive.
