Top Buttons


  • The Select (Square) Button

In this mode one can select objects inside a square shape. You can also move either the selection (if any) by clicking into it or single atom by clicking directly onto them.

  • The Select (Lasso) Button

In this mode one can select objects inside a free form shape. You can also move either the selection (if any) by clicking into it or single atom by clicking directly onto them.

  • The Eraser Button

This mode makes it possible to delete objects. It deletes the current selection and deletes anythign you click on afterwards.

  • The Zoom In/Zoom Out Buttons

This zooms the structure in or out. It only sets the scale of the drawing, it does not change the structure. These buttons have no mode associated.


  • The Undo Button

This button undos the last action. It is only active when something was done before. This button does not have a mode associated.


  • The Redo Button

This button redos the last undone action. It is only active when something was undone before. This button does not have a mode associated.


  • The Cut Button

This button removes the current selection (or all, if none) from the drawing and puts it into the system clipboard. Paste can be used to use it again. JChemPaint uses several formats (SMILES, mol, CML, SVG) so you should also be able to paste it into other programs (depending on abilities of the program). This button does not trigger a mode.


  • The Copy Button

This button copies the current selection (or all, if none) into the system clipboard. Paste can be used to use it again. This button does not trigger a mode.


  • The Paste Button

This button puts the content of the system clipboard into the drawing. The system clipboard must either contain a copy from JChemPaint or text in a format JChemPaint can read. This button does not trigger a mode.


  • The Flip Horizontal/Vertical Buttons

These buttons flip either the selection or the complete structure (if no selection is done) around the horizontal or vertical axis. These buttons do not trigger a mode.


  • The Rotate Button

This button rotates the current selection or the whole structure if nothing is selected. Click and drag round to achieve the effect.
