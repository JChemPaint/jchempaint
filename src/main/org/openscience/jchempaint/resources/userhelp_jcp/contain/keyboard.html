Commands via keyboard

Apart from triggering menu items via the keyboard (shortcuts are given in menus) there are also a number of other convenient keyboard actions:

  • If you hover over an atom, you can change the atom symbol by hitting the symbol key, e. g. O for Oxygen (obviously this only works for single-letter elements).
  • +/- keys zoom in/out. This is like hitting /.
